Projeto de Atividade Extensionista "Emendas em Foco"

O link do repositório acima reúne arquivos do projeto extensionista obrigatório do curso de Ciência de Dados da Uninter. O projeto analisou dados abertos de emendas parlamentares para informar e fortalecer Organizações da Sociedade Civil. Os resultados estão documentados em um Jupyter Notebook disponível no repositório.

Desafio Ciência de Dados "Previsão de preço"

No link acima foi disponibilizado o projeto que foi desenvolvido para o programa Lighthouse da Indicium e inclui a análise exploratória de dados de locações em Nova Iorque e a criação de um modelo de Machine Learning para prever valores de aluguel. O repositório contém instruções para configurar o ambiente, os dados necessários e o modelo treinado disponível para download. O modelo pode ser usado para previsões com normalizações adequadas no input dos dados.
